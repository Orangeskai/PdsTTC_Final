# PdsTTC Kelompok 4

## Game TicTacToe dibuat pada python dan tkinter

## Konsep Singkat
- Permainan
  - Cek progress
    - Selesai
      - Menang
        1. Tampilkan Pesan Menang Dan Quote
        2. Reset Game
      - Kalah
        1. Tampilkan Pesan Kalah Dan Quote
        2. Reset Game
      - Seri
        1. Tampilkan Pesan Seri Dan Quote
        2. Reset Game

## Requirements 
- Python3
- Tkinter
